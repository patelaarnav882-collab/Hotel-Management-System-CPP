Hotel Management System (C++)

Overview

This project is a **Hotel Management System** developed using **C++**. It is a console-based application that helps manage hotel operations like room booking, customer records, and room availability efficiently.

---

Features

* Admin Login System
* Room Booking
* Cancel Booking
* View Room Availability
* Search Customer Details
* File Handling for Data Storage
* Object-Oriented Programming (OOP) Concepts Used

---

Technologies Used

* **Language:** C++
* **Concepts:**

  * Classes & Objects
  * File Handling
  * Functions
  * Conditional Statements

---

Project Structure

```
Hotel-Management-System/
│── project.cpp

│── README.md
```

---

How to Run

1. Install any C++ compiler (like GCC)
2. Open terminal in project folder
3. Compile the code:

   ```
   g++ project.cpp -o hotel
   ```
4. Run the program:

   ```
   ./hotel
   ```
